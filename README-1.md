# 🤖 Mr Cyborg PDF — Neural PDF Engine v4.2

Un éditeur PDF ultra-futuriste avec IA, animations 2D/3D et sous-agents intelligents.

## 🚀 Déploiement sur Vercel

### Méthode 1 — Interface Vercel (recommandé)
1. Créer un compte sur [vercel.com](https://vercel.com)
2. Cliquer **"New Project"** → **"Upload"**
3. Glisser ce dossier (`mr-cyborg-pdf/`)
4. Cliquer **"Deploy"** → C'est en ligne ! ✅

### Méthode 2 — Vercel CLI
```bash
npm install -g vercel
cd mr-cyborg-pdf
vercel deploy
```

### Méthode 3 — GitHub + Vercel
1. Push ce dossier sur GitHub
2. Lier le repo sur vercel.com
3. Auto-déploiement à chaque commit
